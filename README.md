BadmintonHub is a modern, mobile-first Progressive Web Application (PWA) designed to streamline badminton club management. It empowers club admins to schedule sessions and track membership, while providing players with rich statistical insights, match recording, and personalized coaching advice.
Built with React 19 and TypeScript, styled with Tailwind CSS, and powered by Google Gemini for intelligent analytics.
